# HelpHub Client

This package contains the HelpHub frontend application built with React, TypeScript, and Vite.

## Available Scripts

- `npm run dev` – start the Vite development server
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally
- `npm run lint` – run ESLint with the flat config in `eslint.config.js`

## Project Structure

- `src/` – application source code
- `public/` – static assets copied as-is
- `dist/` – generated at build time (ignored in git)

Environment variables for the client live in `client/.env`.
